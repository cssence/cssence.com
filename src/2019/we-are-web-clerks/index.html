We Are Web Clerks

@WeAreWebClerks Vienna 2019

Event
2019-11-25T05:38:00Z

I’ve been attending the first-ever WeAreWebClerks Community Conference 2019 in Vienna. Glad to see one of the most livable cities finally has a down-to-earth (read: not ReactJS-and-the-likes) web developer conference.

Talks & Speakers

  • “Building” by Jeremy Keith
  • “Dark UX patterns” by Lisa Gringl
  • “Rage Against the Content Machine” by Max Böck
  • “Why Every Interface Should Be Black & White” by Heydon Pickering
  • “Stakeholder-Centric Design” by Che Harvey
  • “Does It Work? Using The New CSS Layout” by Rachel Andrew
  • “I’m Just an Old Guard Dev in a New Guard World” by Charlie Owen
  • “Using a Modern Web to Recreate 1980s Horribly Slow and Loud Loading Screens” by Remy Sharp
  • “Emotional, Memorable Design” by Vitaly Friedman

During lunch break, I’ve also been attending these two Lightning Talks:

  • “More ‘Hurrah!’, Less ‘Krawuzzi Kapuzzi’” by Stefan Baumgartner
  • “Testing Web Accessibility” by Adrián Bolonio
