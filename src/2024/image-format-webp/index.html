Friendly reminder, the webp image format is a thing.

I’ve always been using an image compressor to optimize images on my website. Looking at the source code, apparently it has only been July of last year when I started using webp, and I’ve been using it pretty much exclusively ever since.

It comes out superior for both lossy and lossless images. Browser support for webp has been there since what feels like forever.
