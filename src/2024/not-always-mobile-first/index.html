Start with the basics

In Mobile First, you start with the mobile styles and @media (min-width: ‚Ä¶) your way up.

In Basics First, you start with the mundane styles and hide the complex or fine-tuned stuff behind media queries. To do so, ideally you can start with the default the browser gives you out of the box, and enhance from there. As a result, a benefit of Basics First is that there is almost no need for overriding your own styles: Desktop does not need to override Mobile, and vice versa.
