Miriam Suzanne wrote on 2024-02-13T06:22:06Z

Continued from previous comment.

We can’t set accentColor for good reason. That gives us CSS access to the operating system accent color, which can also be set by users.

And we don’t necessarily need CSS access to the accent-color because… we set it. We can set it from a variable, and access that variable if we want.
