CSS Day HTML Special

@CSSDayConf 2016 Day #1

Event
2016-06-16T17:43:00Z

Continental Europe and web development conferences? Well, Amsterdam is the place to be, think no further than CSS Day.

In 2016—for the first time—the actual CSS Day on June 17 was preceded by an HTML Special, with the following amazing speakers.

Talks & Speakers

  • “<a>” by Jeremy Keith
  • “<link>” by Yoav Weiss
  • “<layer>” by Peter-Paul Koch
  • “<source>” by Simon Pieters
  • “<noscript>” by Niels Leenheer
  • “<input>” by Monica Dinculescu
  • “<iframe>” by Peter van der Zee
  • “<html>” by Lea Verou

In case you’ve noticed that all talk titles are HTML elements, it didn’t stop there. Not even the coffee <br>eaks were safe. 😉
